Portfolio

Check out some of my recent projects:

  1. 1. Full-stack web development project: Campified

    Campified is a full-stack web app for discovering and sharing camping sites. Built with Node.js, Express, and MongoDB, it features a modern, user-centric design with a professional navbar, custom dropdowns, and beautiful typography. The backend is secure and efficient, with Helmet and HTML sanitation to prevent attacks, and Passport.js for user authentication. Innovative features include efficient pagination, truncated descriptions, and Mapbox integration for editing campground locations. Personalization options allow users to customize their campgrounds and see their username in the navbar. Campified optimizes image management with Cloudinary and Multer, and is regularly updated and maintained for optimal performance and user satisfaction.

  2. 2. Time Lord's Memory

    Time Lord's Memory is a fun and challenging memory game with visually appealing design and engaging gameplay. Utilizing front-end technologies such as jQuery and jQueryUI libraries, the game's core functionality includes button animations, sound effects, and scorekeeping. It employs advanced features such as asynchronous functions and timeouts to create a seamless gaming experience. The app features a custom-designed modal for game instructions and user interactions using responsive design principles and Bootstrap. Continuously updated with new features and optimizations, Time Lord's Memory showcases my skills in front-end development and my ability to create seamless user experiences.

  3. 3. Colours App

    Colours App is a React web app I developed for colour conversion and manipulation. It features a custom level slider and a dropdown for switching colour modes. To avoid generic Material UI components, I handpicked and customized Node packages to better implement the design plan I had in mind. As the course was with class components, I worked harder to write and understand the extra code. Additionally, the app is responsive, rearranging elements as the screen size changes. Colours App showcases my ability to create a functional, visually-pleasing web app with custom components.

  4. 4. Lights Out Game - Customizable & Responsive

    Lights Out is a customizable and responsive game that I created based on Colt Steele's Lights Out project. Users can specify the number of rows and columns they want to play with, and the game adapts accordingly. The game's interface is clean and user-friendly, providing a seamless experience for players. To build this project, I used React for the front-end and CSS for styling. The game is optimized for various screen sizes, ensuring that users can enjoy it on any device. Overall, Lights Out showcases my ability to create engaging and responsive applications using modern web development tools.

  5. 5. React TODO App with Context, Hooks, and Reducers

    I developed a Todo List application using React, based on Colt Steele's popular project. The app allows users to add, delete, and mark todos as completed. To enhance the user experience, I implemented both light and dark color schemes for the app, making it easier on the eyes. Additionally, I added a "Restore Todos" button that restores two default todos in the app if they are accidentally deleted. The app is fully responsive and can be used on any device. Overall, this project showcases my ability to create a fully-functional web application using React and add custom features to improve the user experience.
