Portfolio

Check out some of my recent projects:

  1. 1. Full-stack web development project: Campified

    Campified is a full-stack web app for discovering and sharing camping sites. It features a modern and professional UI/UX design with custom navigation and layout, adhering to modern design principles for an intriguing user experience. Built using Node.js and Express on the backend, MongoDB for data storage, and EJS templating language for dynamic rendering on the client-side, the backend is secure and efficient, with Helmet and HTML sanitation to prevent attacks, and Passport.js for user authentication. Innovative features include efficient O(logN) database queries for both backward & forward pagination. Personalization options allow users to see and customize their campgrounds. Campified optimizes image management with Cloudinary and Multer, and is regularly updated and maintained.

  2. 2. Time Lord's Memory

    Time Lord's Memory is a fun and challenging memory game with visually appealing design and engaging gameplay. Utilising jQuery and jQueryUI libraries to build the game's core which includes the game progression algorithms alongwith custom animations, scorekeeping, sound effects. Having coded the game from scratch, employed asynchronous functions with a custom timeout function to create a seamless gaming experience. The app features a responsive and adaptable design which scales on all devices, including custom modal for game instructions and info. Continuously updated with new features and optimizations, Time Lord's Memory showcases my skills in front-end development and my ability to create seamless user experiences.

  3. 3. Colours

    'Colours App' allows users to explore and create colour palettes for their web designs. Featuring an eye-pleasing and sleek interface which draws attention, the project employs React Router for front-end routing, Chroma.js for colour conversion and dynamic text luminance. Also, I designed customised components such as a slider for changing colour intensity and a dropdown for switching colour formats which make the application stand out. Adhering to modern coding and design practices, I prioritised the intuitiveness of the UI and made sure that using the app is worthwhile on every device.

  4. 4. Lights Out

    Lights Out is an intriguing and customisable puzzle game that I created based on React. Users can specify the number of rows and columns they want to play with, and the game adapts accordingly. The game's interface is modern and captivating, providing a seamless experience for any player. Implemented the game logic with a reset option, optimising it for various screen sizes, ensuring that users can enjoy it on any device.

  5. 5. React todo

    I developed a Todo List application using React. Utilising React Hooks, Context and Reducers, the app allows users to add, delete, and mark todos as completed. To enhance the user experience, I implemented both light and dark color schemes for the app, making it easier on the eyes. Additionally, I added a "Restore Todos" button that restores two default todos in the app if they are accidentally deleted. The app is fully responsive and can be used on any device.
