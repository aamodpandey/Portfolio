Portfolio

Check out some of my recent projects:

  1. Full-stack web development project: YelpCamp

    YelpCamp, inspired by Colt Steele's project, is a full-stack web application developed by me. I enhanced its aesthetics, performance, and features, including validation checks on the server side, pagination using indexing in MongoDB, and restrictions on image size and number of uploads. I also implemented a more aesthetically pleasing layout using MUI CSS, ensured responsiveness, and added security measures to make it robust.

  2. SimonXDoctorWho - A Creative Twist to the Classic Memory Game

    I built SimonXDoctorWho from scratch using HTML, CSS, JavaScript, and jQuery. As a fan of Doctor Who, I added Doctor Who-themed elements such as TARDIS sound and animation, along with images from the show. I improved the game's aesthetics and made it touch-friendly on mobile devices. I also implemented features such as game over, high score, and restart. Overall, SimonXDoctorWho was a fun challenge that allowed me to showcase my creativity and programming skills.

  3. React TODO App with Context, Hooks, and Reducers

    This React TODO app was built using functional components and utilizes the power of context, hooks, and reducers. Inspired by Colt Steele's React course, I enhanced the app to include a dark mode, a restore todos button, and local storage persistence, allowing changes to persist even after reloads. With a clean and intuitive interface, this app showcases my proficiency in React development and my ability to integrate multiple features into a single project.

  4. Lights Out Game - Customizable & Responsive

    Inspired by Colt Steele's Lights Out project, I built a customizable and responsive version of the game. Players can manually enter the number of rows and columns they want to play with, and the game adjusts accordingly. The layout is responsive and visually appealing, making for a great user experience. This project was built using React and styled with CSS.

  5. Color Picker App

    A colorful web app created using React, which helps you select and preview colors with ease. Inspired by my love for design and creativity, and built using the React framework, this app allows you to easily select and copy the HEX or RGB values of any color you like. The app also includes a built-in color palette for your convenience. This project is a testament to my skills as a Full-Stack Web Developer and showcases my proficiency in working with React.
